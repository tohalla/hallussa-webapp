Sales pitch

Have you noticed appliances around you being broken for a long time? Have you tried to report but noticed that you should call somewhere but only withing working hours and you don't happend to be free at those times? Our product solves this problem by giving users easy way to report a malfunctioning appliance at any hour. The organisation will be informed instantly and the work is delegated to the right worker. You can get notification when the appliance is working again.

List of features

    For end users:

  • Easy and fast way to report malfunction appliances
  • Report at any hour and no need to call anywhere
  • Instant notification for the appliance's organisation
  • Get notified when work is done

    • For organisations:

  • Simple and easy to use user interface
  • See the performance of maintainers
  • Add and edit appliances and maintainers information
